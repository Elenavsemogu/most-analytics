# Проект: Аналитика для канала MOST

## Мета
- **Категория:** gambling
- **Статус:** deployed + auto-collect
- **Приоритет:** high
- **Последнее обновление:** 2026-03-02

## Описание
Веб-дашборд аналитики для Telegram-канала @mostcareer. Автоматический ежедневный сбор данных, классификация контента, GPT-4o анализ с привязкой к маркетинговой стратегии.

## Продакшн
- **URL:** https://most-analytics.onrender.com
- **Пароль:** most2026
- **GitHub:** Elenavsemogu/most-analytics
- **Хостинг:** Render (auto-deploy из main)
- **Cron:** ежедневно 08:00 UTC (TGStat метрики + scrape новых постов)

## Стек
- **FastAPI + Uvicorn** — бэкенд
- **SQLite** — хранение данных, снимков, анализов
- **TGStat API** — метрики канала (подписчики, ERR, охват, CI, ER)
- **Telegram t.me/s/ scraper** — посты (серверный + браузерный fallback)
- **OpenAI GPT-4o** — глубокий анализ с привязкой к стратегии
- **Chart.js** — графики: TGStat-стиль + умные (контент-микс, дни, стратегия)
- **Render** — хостинг + cron
