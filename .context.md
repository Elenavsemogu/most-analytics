## Описание
Аналитическая система для Telegram-канала MOST: TGStat API + OpenAI GPT + n8n. Еженедельный автоматический отчёт с доставкой в Telegram.

## Стек
- **n8n** — оркестрация (локально через `npx n8n`)
- **TGStat API** — данные канала, посты, метрики
- **OpenAI GPT-4o-mini** — анализ данных и генерация инсайтов
- **Telegram Bot API** — доставка отчётов
